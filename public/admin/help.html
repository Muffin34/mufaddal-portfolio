Portfolio Admin Guide

Use this page as your quick map for editing text, adding YouTube videos, uploading media, and publishing changes.

Open Admin Editor View Live Site

Edit Homepage

  1. Open Home Page.
  2. Click Edit Home Page.
  3. Change hero text, metrics, client logos, selected work, experience, skills, education, or resume link.
  4. Click Save, then Publish.

Add A YouTube Video

  1. Open Work Page + Videos.
  2. Click Edit Work Page.
  3. Choose E-Learning Videos or Broadcast / Promo Videos.
  4. Click Add.
  5. Paste only the YouTube video ID, not the full link.
  6. Add a title, module/project code, and category or tag.
  7. Click Save, then Publish.
Example: from https://www.youtube.com/watch?v=QYtqpC3wWi0, paste only QYtqpC3wWi0.

What To Edit Where

Home Page

  • Main headline and intro paragraph
  • Resume link, contact email, Linktree link
  • Homepage metrics, client logo marquee, and selected work cards
  • Experience, skills, software, AI tools, education

Work Page + Videos

  • E-learning video gallery
  • Broadcast and promo video gallery
  • Workflow steps, stats, and tools
  • Project descriptions and categories

Publishing

After you click Publish, Decap commits the change to GitHub. Netlify then rebuilds the site automatically. The live site usually updates in 1-3 minutes.
