Best way to add a project

  1. Upload the finished video to YouTube.
  2. Copy the video ID from the URL.
  3. Open Add YouTube Video.
  4. Add the title, module/project code, and category or tag.
  5. Save and Publish.

Finding the YouTube ID

From this link:

https://www.youtube.com/watch?v=QYtqpC3wWi0

Paste only this part into the admin:

QYtqpC3wWi0

Do not paste the full YouTube link into the video ID field. The website builds the thumbnail and player from the ID automatically.
